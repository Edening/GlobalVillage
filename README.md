# vue-quasar-supermapwebgl

> vue+supermap iclent3d for webgl演示程序

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:8888
$ npm run dev

# build for production with minification
$ npm run build

# lint code
$ quasar lint

# run dist
$ npm run dist
```
